# [WIP] Shifter SDK for Node.js

## Getting started

```
$ git clone https://github.com/megumiteam/shifter-sdk-for-javascript.git
$ cd shifter-sdk-for-javascript
$ npm i
```

## Contribution

### Type

```
$ npm run type
```

### Lint

```
$ npm run lint
or
$ npm run lint -- --fix
```

### Test

```
$ npm test
$ npm test -- --watch
```
